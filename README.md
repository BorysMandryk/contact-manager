# Contact Manager
## Server
1. Move to server/ContactManager
```
cd server/ContactManager
```
2. Build application:
```
dotnet build
```
3. Run:
```
dotnet run --project Web
```

## Client:
1. Move to client:
```
cd client
```
1. Install packages:
```
npm install
```
2. Run development server:
```
ng serve
```
or 
```
npm start
```
